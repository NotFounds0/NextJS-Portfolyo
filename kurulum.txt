github repo kullanmak isterseniz ilk önce bir token oluşturmalısınız.
Personal Access Token Oluşturma:
GitHub hesabına giriş yap.
Sağ üst köşede profil simgesine tıkla ve "Settings" (Ayarlar) seçeneğine git.
Sol menüden "Developer settings" bölümüne git.
"Personal Access Tokens" seçeneğine tıkla ve yeni bir token oluştur.
Bu token'ı, sadece ihtiyaç duyduğun izinlerle oluştur.
sadece git repo seçmeniz yeterli olucaktır.
